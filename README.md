# AI Customer Support Ticketing System

An AI-powered customer support ticket management platform built using:

- Next.js
- Node.js
- Express.js
- MongoDB
- Python
- Scikit-learn

---

## Features

- AI Ticket Classification
- Automatic Ticket Routing
- SLA Tracking
- Agent Dashboard
- Admin Dashboard
- Customer Dashboard
- Customer Satisfaction Analytics
- JWT Authentication
- File Upload
- Realtime Notifications

---

## Tech Stack

### Frontend

- Next.js
- Tailwind CSS
- React Query

### Backend

- Express.js
- MongoDB
- JWT

### AI

- Python
- Flask
- Scikit-learn
